# Anime-Nuvio
Compiled Anime Plugins with Multi Language Support.

## Installation

1. Open **Nuvio** app
2. Go to **Settings вЖТ Local Scrapers**
3. Add this repository URL:
   ```
   https://raw.githubusercontent.com/D3adlyRocket/Anime-Nuvio/refs/heads/main/manifest.json
   ```
4. Enable the scrapers you want to use
